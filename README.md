# hooyantsing-resume

> 公开简历。

### 构建简历

1. 编辑简历信息，目录 `src/resume/*`；
2. 安装依赖，命令 `yarn`；
3. 预览模式，命令 `yarn serve`；
4. 导出PDF，命令 `yarn export`，导出至 `export` 目录。
